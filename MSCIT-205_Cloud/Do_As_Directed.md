### Do As Directed / Definitions
13. **Define: Cloud Computing** - On-demand delivery of compute power, database, storage, applications, and other IT resources via the internet with pay-as-you-go pricing. (Aug 2022)
14. **What is peer-to-peer?** A distributed application architecture that partitions tasks or workloads between peers. (Aug 2022, July 2025)
15. **Define: Data Redundancy** - Storing the same data in multiple locations to ensure it is not lost if a server fails. (Aug 2022, Jan 2023)
16. **What is Symmetric Key Encryption?** Using a single key for both encryption and decryption. (Aug 2022, July 2025)
17. **What is Asymmetric Encryption?** Using a pair of keys (public and private) for encryption and decryption. (Aug 2022)
18. **SAML stands for:** Security Assertion Markup Language. (Jan 2023, July 2023)
19. **CSB stands for:** Cloud Services Brokerage. (Jan 2023, July 2023)
20. **What is vendor-locking?** A situation where a customer is dependent on a single cloud provider and cannot easily move to another. (Jan 2023)
21. **What is Data segregation?** Isolating a customer's data from other customers' data in a shared environment. (Jan 2023)
22. **What is NoSQL?** "Not only SQL" - a non-relational database that stores and retrieves data in formats other than tabular relations. (Jan 2023)
23. **What is Server virtualization?** Masking server resources (including the number and identity of individual physical servers) from server users. (Jan 2023)

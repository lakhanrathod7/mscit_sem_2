# MSCIT-205: Cloud Infrastructure and Services
## Important MCQs from Previous Papers

1. SOA stands for ________. (Aug 2022)
   A. Service Oriented Architecture B. System Oriented Architecture C. Structure Oriented Architecture D. Security Oriented Architecture
   Answer: A.
   SOA is a style of software design where services are provided to components.

2. VMM stands for ________. (Aug 2022)
   A. Virtual Machine Mode B. Virtual Machine Monitor C. Virtual Machine Management D. Virtual Machine Master
   Answer: B.
   VMM is another name for a Hypervisor.

3. ________ providers give low-level abstraction of physical device. (Aug 2022)
   A. SaaS B. PaaS C. IaaS D. BaaS
   Answer: C.
   IaaS (Infrastructure as a Service) provides virtualized computing resources.

4. Private cloud includes ________. (Aug 2022)
   A. Virtualization B. Storage C. Network D. All of Above
   Answer: D.
   Private clouds require a full infrastructure stack dedicated to one entity.

5. UML stands for ________. (Aug 2022)
   A. User–Machine Linux B. User–Management Linux C. User–Master Linux D. User–Mode Linux
   Answer: D.
   In cloud context, UML often refers to User-Mode Linux (virtualization).

6. SaaS includes ________. (Aug 2022)
   A. Cloud Programming B. Cloud Hosting C. Cloud Application D. All of Above
   Answer: C.
   SaaS delivers software applications over the internet.

7. GCE stands for ________. (Aug 2022)
   A. Google Cloud Engine B. Google Compute Engine C. Google Computer Engine D. Google Compiler Engine
   Answer: B.
   GCE is Google's Infrastructure-as-a-Service (IaaS) offering.

8. ________ react promptly & appropriately to any security violence incident. (Aug 2022)
   A. Detective Control B. Preventive Control C. Corrective Control D. Deterrent Control
   Answer: C.
   Corrective controls aim to repair damage after an incident.

9. IDS stands for ________. (Aug 2022)
   A. Intrusion Detection System B. Intrusion Detection Service C. Intrusion Detection Security D. Intrusion Detection Structure
   Answer: A.
   Intrusions are monitored by an IDS.

10. TLS stands for ________. (Aug 2022)
    A. Transport Level Security B. Transport Layer Security C. Transport Local Security D. Transport List Security
    Answer: B.
    TLS is a cryptographic protocol for secure communication.

11. _____ charged for the shipped data before compression. (Jan 2023)
    A. Data Write B. Data Shipping C. Data Sniffing D. All the Above
    Answer: B.
    Data shipping refers to the cost of moving data between systems.

12. The cost of setup of a cloud service is __________ type of Cost. (Jan 2023)
    A. Indirect Cost B. Individual Cost C. Direct Cost D. None of these
    Answer: C.
    Direct costs are specifically attributable to a service.

13. Full form of SDLC_______. (Jan 2023)
    A. Software Development Life cycle B. System Design Life cycle C. System Development Life cycle D. None of these
    Answer: A.
    SDLC is a process for planning, creating, testing, and deploying systems.

14. _______ is a new approach to access large computational power, huge storage, and a variety of services. (Jan 2023)
    A. Cluster B. Mainframe C. Grid Computing D. Database System
    Answer: C.
    Grid computing pools resources from multiple locations.

15. Which one of the following options can be considered as the Cloud. (Jan 2023)
    A. Internet B. Web Application C. Hadoop D. All the above
    Answer: D.
    These all contribute to or utilize the cloud ecosystem.

16. Which of the following is an essential concept related to Cloud? (Jan 2023)
    A. Abstraction B. Reliability C. Productivity D. None of these
    Answer: A.
    Cloud abstracts underlying hardware from users.

17. Which of the following has many features of that is now known as cloud computing? (Jan 2023)
    A. Web services B. Software C. Internet D. All the above
    Answer: D.
    Cloud computing evolved from these technologies.

18. ______ is the example of IAAS. (Jan 2023)
    A. Microsoft Azure B. Amazon Web Service C. Google Compute Engine D. All the Above
    Answer: D.
    All three major providers offer IaaS.

19. Which of the following is the most important area of concern in cloud Computing. (Jan 2023)
    A. Security B. Storage C. Scalability D. None of these
    Answer: A.
    Security is consistently the top concern for cloud adopters.

20. _____ Refers to the location and management of cloud’s Infrastructure. (Jan 2023)
    A. Service B. Deployment C. Application D. All of these
    Answer: B.
    Deployment models (Public, Private, etc.) define the setup.

21. Amazon Beanstalk and Google App Engines are example of ___________. (July 2023)
    A. IaaS B. PaaS C. SaaS D. DBaaS
    Answer: B.
    PaaS provides a platform for developers to deploy applications.

22. ___________ Virtualization uses binary translation and direct approach as a technique for operations. (July 2023)
    A. Full B. Para C. Hypervisor D. Hardware Assisted
    Answer: A.
    Full virtualization allows an unmodified guest OS to run.

23. __________ Control Reduce attack on cloud system by giving a warning sign. (July 2023)
    A. Detective B. Corrective C. Preventive D. Deterrent
    Answer: D.
    Deterrent controls discourage attackers (like warning signs).

24. A ____________ cloud offers infrastructure and services off-site. (July 2023)
    A. Private B. Hybrid C. Public D. None of them
    Answer: C.
    Public clouds are owned and operated by third-party providers.

25. __________________ is a type of model for RaaS . (July 2023)
    A. Managed B. Assisted C. Self Service D. All of them
    Answer: D.
    Recovery-as-a-Service can be delivered in several ways.

26. In __________ scaling, instead of adding or removing resources the capacity of virtual machine is changed. (July 2023)
    A. Vertical B. Horizontal C. Auto D. Manual
    Answer: A.
    Vertical scaling (scaling up/down) means resizing a single resource.

27. In SOA _____________ is used for web service implementations (July 2023)
    A. SMTP B. SOAP C. REST D. WSDL
    Answer: B.
    SOAP is a common protocol for SOA web services.

28. ____________ protocol is cryptographically secure protocol used to provide integrity during TCP/IP communication. (July 2023)
    A. SSL B. TSL C. HTTPS D. Both A and B
    Answer: D.
    Both SSL and its successor TLS provide security.

29. _____________ provider offers an actual working software application to clients. (July 2023)
    A. Paas B. IaaS C. SaaS D. None of them
    Answer: C.
    SaaS (Software as a Service) provides end-user applications.

30. ______ intercepts all the data and I/O between customer data centre and public clouds. (July 2023)
    A. Cloud Storage Gateway B. Cloud Data Gateway C. Cloud Security Gateway D. Cloud Integrity Gateway
    Answer: A.
    Gateways bridge on-premise storage with cloud storage.

31. Which organization introduced the most accepted definition of cloud computing? (July 2025)
    A. ISO B. IEEE C. NIST D. IETF
    Answer: C.
    NIST (National Institute of Standards and Technology) provides the standard definition.

32. Which technology enables cloud computing by separating hardware from software? (July 2025)
    A. Grid Computing B. Virtualization C. Parallel Processing D. Data Warehousing
    Answer: B.
    Virtualization allows multiple VMs to share physical hardware.

33. Which cloud service model provides hardware infrastructure as a service? (July 2025)
    A. IaaS B. SaaS C. PaaS D. DBaaS
    Answer: A.
    IaaS (Infrastructure as a Service) is the foundational hardware layer.

34. Google App Engine is an example of: (July 2025)
    A. IaaS B. SaaS C. SECaaS D. PaaS
    Answer: D.
    GAE is a Platform as a Service (PaaS).

35. Which feature of cloud computing refers to automatic scaling of resources? (July 2025)
    A. Resource pooling B. Rapid elasticity C. Broad network access D. Measured service
    Answer: B.
    Elasticity is the ability to scale resources on the fly.

36. Which is a major risk when migrating data to the cloud? (July 2025)
    A. Data incompatibility B. Reduced bandwidth C. High latency D. Increased local storage
    Answer: C.
    Latency (delay) can be a significant issue for cloud-hosted data.

37. Which cloud type provides maximum control and security? (July 2025)
    A. Public cloud B. Hybrid cloud C. Private cloud D. Community cloud
    Answer: C.
    Private clouds offer the most dedicated control and security.

38. Which of the following is not an advantage of public cloud? (July 2025)
    A. Scalability B. Location independence C. Low cost D. High security
    Answer: D.
    While secure, public clouds are generally perceived as less secure than private ones.

39. Which cloud component manages and monitors virtual machines? (July 2025)
    A. Hypervisor B. Router C. Middleware D. Firewall
    Answer: A.
    The hypervisor (VMM) controls the virtual machine environment.

40. Which of the following is a challenge in cloud computing adoption? (July 2025)
    A. Cost savings B. Reduced time to market C. Insecure APIs D. Elastic resource allocation
    Answer: C.
    Security of APIs is a major technical challenge for cloud users.

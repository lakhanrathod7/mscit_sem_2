# MSCIT-205: Cloud Infrastructure and Services
## Important MCQs from Previous Papers

1. **SOA stands for ________.** (Aug 2022)
   A. Service Oriented Architecture B. System Oriented Architecture C. Structure Oriented Architecture D. Security Oriented Architecture
   **Answer: A.**

2. **VMM stands for ________.** (Aug 2022)
   A. Virtual Machine Mode B. Virtual Machine Monitor C. Virtual Machine Management D. Virtual Machine Master
   **Answer: B.**

3. **________ providers give low-level abstraction of physical devices.** (Aug 2022, July 2025)
   A. SaaS B. PaaS C. IaaS D. BaaS
   **Answer: C.** (Infrastructure as a Service).

4. **UML stands for ________.** (Aug 2022)
   A. User–Machine Linux B. User–Management Linux C. User–Master Linux D. User–Mode Linux
   **Answer: D.** (Note: In MSCIT-205 context, it refers to User-Mode Linux, not the modeling language).

5. **GCE stands for ________.** (Aug 2022)
   A. Google Cloud Engine B. Google Compute Engine C. Google Computer Engine D. Google Compiler Engine
   **Answer: B.**

6. **________ react promptly & appropriately to any security violence incident.** (Aug 2022)
   A. Detective Control B. Preventive Control C. Corrective Control D. Deterrent Control
   **Answer: C.**

7. **IDS stands for ________.** (Aug 2022)
   A. Instruction Detection System B. Instruction Detection Service C. Instruction Detection Security D. Instruction Detection Structure
   **Answer: A.** (Intrusion Detection System is also common, but the paper says Instruction).

8. **TLS stands for ________.** (Aug 2022)
   A. Transport Level Security B. Transport Layer Security C. Transport Local Security D. Transport List Security
   **Answer: B.**

9. **_____ charged for the shipped data before compression.** (Jan 2023)
   A. Data Write B. Data Shipping C. Data Sniffing D. All
   **Answer: B.**

10. **Full form of SDLC _______.** (Jan 2023)
    A. Software Development Life cycle B. System Design Life cycle C. System Development Life cycle D. None
    **Answer: A.**

11. **_______ is a new approach to access large computational power, huge storage, and a variety of services.** (Jan 2023)
    A. Cluster B. Mainframe C. Grid Computing D. Database System
    **Answer: C.**

12. **Which of the following is the most important area of concern in cloud computing?** (Jan 2023)
    A. Security B. Storage C. Scalability D. None
    **Answer: A.**

13. **Amazon Beanstalk and Google App Engine are examples of ___________.** (July 2023, July 2025)
    A. IaaS B. PaaS C. SaaS D. DBaaS
    **Answer: B.**

14. **___________ Virtualization uses binary translation and direct approach as a technique for operations.** (July 2023)
    A. Full B. Para C. Hypervisor D. Hardware Assisted
    **Answer: A.**

15. **In __________ scaling, instead of adding/removing resources, the capacity of the virtual machine is changed.** (July 2023)
    A. Vertical B. Horizontal C. Auto D. Manual
    **Answer: A.**

16. **Which organization introduced the most accepted definition of cloud computing?** (July 2025)
    A. ISO B. IEEE C. NIST D. IETF
    **Answer: C.**

17. **Which technology enables cloud computing by separating hardware from software?** (July 2025)
    A. Grid Computing B. Virtualization C. Parallel Processing D. Data Warehousing
    **Answer: B.**

18. **Which feature of cloud computing refers to automatic scaling of resources?** (July 2025)
    A. Resource pooling B. Rapid elasticity C. Broad network access D. Measured service
    **Answer: B.**

19. **Which cloud type provides maximum control and security?** (July 2025)
    A. Public cloud B. Hybrid cloud C. Private cloud D. Community cloud
    **Answer: C.**

20. **Which cloud component manages and monitors virtual machines?** (July 2025)
    A. Hypervisor B. Router C. Middleware D. Firewall
    **Answer: A.**

# MSCIT-201: Object Oriented Concepts and Programming (Java)
## Important MCQs from Previous Papers

1. **Java is a product of ___________.** (Aug 2022)
   A. Microsoft B. Open Source C. Sun Microsystems D. Adobe
   **Answer: C.** Originally developed by Sun Microsystems.

2. **.java file is compiled by ________ and converted into .class file.** (Aug 2022)
   A. Java compiler B. JVM C. Interpreter D. None
   **Answer: A.** The `javac` compiler generates bytecode (.class).

3. **From the given below options, which is NOT a valid identifier in java?** (Aug 2022)
   A. Abc B. 123abc C. _abc123 D. Abc123
   **Answer: B.** Identifiers cannot start with a digit.

4. **Method finalize() is of ________ class.** (Aug 2022)
   A. java.lan.object B. java.awt C. java.lang.Object D. java.util
   **Answer: C.** It's a method of the base Object class.

5. **Two functions in the same class, having same name but different number of arguments is an example of ________.** (Aug 2022)
   A. Method overriding B. Method overloading C. Abstract method D. Constructor
   **Answer: B.** Method overloading allows same name with different parameters.

6. **_______ variables of the class should have only one copy, which is shared among all the objects of a class.** (Aug 2022)
   A. Instance B. Object C. Private D. Static
   **Answer: D.** Static variables are class-level.

7. **Vector class is available in _________ package.** (Aug 2022, July 2025)
   A. java.lang B. java.util C. java.awt D. java.thread
   **Answer: B.** Part of the Java Utility package.

8. **A class which is not instantiated is called _________ class.** (Aug 2022)
   A. Base class B. Abstract class C. Derived class D. Super class
   **Answer: B.** Abstract classes cannot be instantiated.

9. **If a class that implements an interface does not implement all the methods of the interface, then the class becomes a/an ……………………. class.** (Aug 2022)
   A. abstract B. final C. static D. super
   **Answer: A.** It must be declared abstract.

10. **Variables with _______ access specifiers of a class can be accessible by that class and derived class but not by outsiders.** (Aug 2022)
    A. Public B. Private C. Protected D. All
    **Answer: C.** Protected provides access to subclasses and package members.

11. **An object is an instance of a __________** (Jan 2023)
    A. A class B. A method C. An object D. A data field
    **Answer: A.**

12. **The keyword __________ is required to declare a class.** (Jan 2023)
    A. Public B. Class C. Private D. All
    **Answer: B.** `class` keyword is mandatory.

13. **________ is invoked to create an object.** (Jan 2023)
    A. A constructor B. The main method C. A method with return type D. void method
    **Answer: A.** Constructors initialize new objects.

14. **Which of the following is an invalid declaration for the main method?** (Jan 2023)
    A. public static void main (String [ ] args)
    B. public static strictfp void main(String args[ ])
    C. final static public void main (String args[ ])
    D. All are Correct
    **Answer: D.** All these variations are valid.

15. **A top level class may have which one of the following access modifiers?** (Jan 2023)
    A. Package B. Public C. Private D. Protected
    **Answer: B.** Top-level classes can be public or package-private (default).

16. **A subclass is also called as___________** (Jan 2023)
    A. inner class B. nested class C. derived class D. hidden class
    **Answer: C.**

17. **Attribute of an object can include information about_____** (Jan 2023)
    A. state B. method C. Behavior D. procedures
    **Answer: A.** Attributes represent state, while methods represent behavior.

18. **The methods defined by Thread class that help to manage threads are:** (Jan 2023)
    A. join B. getName C. stop D. all
    **Answer: D.**

19. **A default constructor has_________** (Jan 2023)
    A. no arguments B. one argument C. no return type D. Two arguments
    **Answer: A.** Default constructor takes no parameters.

20. **What software runs a Java program on a computer?** (July 2023)
    A. JRE B. JDK C. Command Prompt D. None
    **Answer: A.** Java Runtime Environment is needed for execution.

21. **Which keyword is used to declare a Java package?** (July 2023)
    A. pkg B. package C. pkge D. None
    **Answer: B.**

22. **All Interface methods in Java are ____ by default.** (July 2023)
    A. public B. abstract C. public and abstract D. None
    **Answer: C.**

23. **Which of the following is NOT a feature of Java?** (July 2025)
    A. Platform-independent B. Secure C. Manual memory management D. Object-oriented
    **Answer: C.** Java has automatic garbage collection.

24. **Which component converts Java source code into bytecode?** (July 2025)
    A. javac compiler B. JVM C. JRE D. Interpreter
    **Answer: A.** `javac` compiles .java to .class (bytecode).

25. **Which method is automatically called before an object is garbage collected?** (July 2025)
    A. destructor() B. delete() C. free() D. finalize()
    **Answer: D.**

26. **Which collection class implements a growable array?** (July 2025)
    A. Array B. Vector C. LinkedList D. HashSet
    **Answer: B.** Vector (and ArrayList) are growable arrays.

27. **Which method is used to convert a String to an int?** (July 2025)
    A. Integer.valueOf() B. String.toInt() C. parse() D. Integer.parseInt()
    **Answer: D.**

28. **Which of the following streams are used to read character data?** (July 2025)
    A. FileReader B. FileInputStream C. FileOutputStream D. DataInputStream
    **Answer: A.** Reader classes handle characters; Stream classes handle bytes.

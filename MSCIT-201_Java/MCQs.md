# MSCIT-201: Object Oriented Concepts and Programming (Java)
## Important MCQs from Previous Papers

1. Java is a product of ___________. (Aug 2022)
   A. Microsoft B. Open Source C. Sun Microsystems D. Adobe
   Answer: C.
   Originally developed by Sun Microsystems.

2. .java file is compiled by ________ and converted into .class file. (Aug 2022)
   A. Java compiler B. JVM C. Interpreter D. None of the above
   Answer: A.
   The javac tool compiles source code into bytecode.

3. From the given below options, which not a valid identifier in java. (Aug 2022)
   A. Abc B. 123abc C. _abc123 D. Abc123
   Answer: B.
   Identifiers cannot start with a digit.

4. Method finalize() is of ________ class. (Aug 2022)
   A. java.lan.object B. java.awt C. java.lang D. java.util
   Answer: C.
   It is a method of the java.lang.Object class.

5. Two functions in the same class, having same name but different number of arguments is an example of ________. (Aug 2022)
   A. Method overriding B. Method overloading C. Abstract method D. Constructor
   Answer: B.
   Overloading happens with different signatures in the same class.

6. _______ variables of the class should have only one copy, which is shared among all the objects of a class. (Aug 2022)
   A. Instance B. Object C. Private D. Static
   Answer: D.
   Static variables belong to the class rather than instances.

7. Vector class is available in _________ package. (Aug 2022)
   A. java.lang B. java.util C. java.awt D. java.thread
   Answer: B.
   It is part of the Java Collection Framework in java.util.

8. A class which is not instantiated is called _________ class. (Aug 2022)
   A. Base class B. Abstract class C. Derived class D. Super class
   Answer: B.
   Abstract classes cannot be used to create objects directly.

9. If a class that implements an interface does not implement all the methods of the interface, then the class becomes a/an ……………………. class. (Aug 2022)
   A. abstract B. final C. static D. super
   Answer: A.
   It must be declared abstract as it contains unimplemented methods.

10. Variables with _______ access specifiers of a class, can be accessible by that class and derived class but they are not be accessible by an outsider. (Aug 2022)
    A. Public B. Private C. Protected D. All of the above
    Answer: C.
    Protected visibility includes subclasses.

11. An object is an instance of a __________ (Jan 2023)
    A. A class B. A method C. An object D. A data field
    Answer: A.
    A class serves as a blueprint for objects.

12. The keyword __________ is required to declare a class. (Jan 2023)
    A. Public B. Class C. Private D. All of above
    Answer: B.
    The 'class' keyword is mandatory for class definition.

13. ________ is invoked to create an object. (Jan 2023)
    A. A constructor B. The main method C. A method with a return type D. A method with a void return type
    Answer: A.
    Constructors initialize the new object.

14. Which of the following is an invalid declaration for the main method in java program? (Jan 2023)
    A. public static void main (String [ ] args) B. public static strictfp void main(String args[ ]) C. final static public void main (String args[ ]) D. All are Correct
    Answer: D.
    All listed variations are technically valid in Java.

15. A top level class may have which one of the following access modifiers? (Jan 2023)
    A. Package B. Public C. Private D. Protected
    Answer: B.
    Only 'public' or default (package) are allowed for top-level classes.

16. A subclass is also called as___________ (Jan 2023)
    A. inner class B. nested class C. derived class D. hidden class
    Answer: C.
    It is a class that inherits from a superclass.

17. Attribute of an object can include information about_____ (Jan 2023)
    A. state B. method C. Behavior D. procedures

    Attributes define the state of an object.

18. Object oriented programming method is followed in________ (Jan 2023)

    Both C# and C++ are object-oriented.

19. The methods defined by Thread class that help to manage threads are : (Jan 2023)
    A. joined B. Get Name C. stop D. all of the above

    Thread class contains these and many more management methods.

20. A default constructor has_________ (Jan 2023)
    A. has no arguments B. has one argument but no return type C. has no return type D. has two arguments
    Answer: A.
    It is a no-argument constructor provided by Java.

21. What software runs a Java program on a computer? (July 2023)
    A. JRE B. JDK C. Command Prompt D. None of these
    Answer: A.
    Java Runtime Environment provides the necessary libraries and JVM.

22. What is the need to mention "static" before main method? (July 2023)
    A. To call main method without creating an object of class B. To make main method as class method common to all instances C. Both A and B D. None of these
    Answer: C.
    Static allows the JVM to call main without instantiation.

23. An object is created at __ time in Java. (July 2023)
    A. Compile-time B. Run time C. Assembling time D. None of these

24. The keyword used to declare a Java package is ____. (July 2023)

25. A Java Class inherits Constants and Methods of an Interface using ____ keyword. (July 2023)

27. To stop or block inheriting a given class, the ___ keyword is used before the class. (July 2023)

28. Creating an object from a class is also called ____. (July 2023)

29. What is the use of Access modifier "public" in Java language? (July 2023)

30. Which are the keywords used to handle exceptions in Java? (July 2023)

32. Which component converts Java source code into bytecode? (July 2025)

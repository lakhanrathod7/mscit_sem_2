### Definitions / Short Answers
24. **What is Robust?** Java is robust because it has strong memory management and lacks pointers that cause memory leaks. (Jan 2023)
25. **What is Multithreading?** The ability of a CPU to provide multiple threads of execution concurrently. (Jan 2023)
26. **What is finalize() method?** A method called by the garbage collector on an object when garbage collection determines that there are no more references to the object. (July 2023)
27. **What are Wrapper classes?** Classes that allow primitive types to be accessed as objects (e.g., Integer for int). (Jan 2023)
28. **Define 'this' keyword.** A reference variable that refers to the current object. (July 2023)
29. **MIN_PRIORITY and MAX_PRIORITY values:** MIN = 1, MAX = 10. (July 2023)
30. **Checked vs Unchecked exceptions:** Checked are verified at compile-time; Unchecked are at run-time. (July 2023)
31. **Who is the inventor of Java?** James Gosling at Sun Microsystems. (July 2023)

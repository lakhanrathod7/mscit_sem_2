# SOLUTIONS (EXAM FORMAT)

## SECTION A: LONG ANSWERS

### Q1. Networking & Components
**Definition:** A computer network is a system of interconnected computers that share resources and information.

**Five Key Components:**
1.  **Sender:** The device that initiates the data transfer (e.g., PC, Mobile).
2.  **Receiver:** The device that receives the message.
3.  **Message:** The actual data being sent (Text, Audio, Video).
4.  **Transmission Medium:** The physical or wireless path (Cables, Radio waves).
5.  **Protocol:** The set of rules that govern data communication (e.g., TCP/IP).

---

### Q2. OSI Reference Model (7 Layers)
1.  **Physical:** Bits, hardware, cables.
2.  **Data Link:** Frames, physical addressing (MAC), error detection.
3.  **Network:** Packets, logical addressing (IP), routing.
4.  **Transport:** End-to-end reliability, flow control (TCP/UDP).
5.  **Session:** Manages dialogue and synchronization.
6.  **Presentation:** Data encryption, compression, and formatting.
7.  **Application:** Network services for users (HTTP, FTP).
*   **Mnemonic:** **A**ll **P**eople **S**eem **T**o **N**eed **D**ata **P**rocessing.

---

### Q4. Network Topologies
*   **Star:** Nodes connect to a central hub. *Advantage:* Easy to install and manage.
*   **Bus:** All nodes connect to one backbone cable. *Advantage:* Very cheap.
*   **Ring:** Each node has exactly two neighbors. *Advantage:* No central controller needed.
*   **Mesh:** Every node connects to every other node. *Advantage:* Highly reliable.

## SECTION B: SHORT ANSWERS

### Q1. IPv4 vs IPv6
*   **IPv4:** 32-bit address, decimal format (e.g., 192.168.1.1), provides ~4.3 billion addresses.
*   **IPv6:** 128-bit address, hexadecimal format, provides almost unlimited addresses (solving exhaustion).

### Q2. Hub, Switch, and Router
*   **Hub:** Layer 1, broadcasts data to all ports (Dumb device).
*   **Switch:** Layer 2, forwards data based on MAC address (Intelligent).
*   **Router:** Layer 3, connects different networks using IP (Smartest).

### Q3. Throughput vs Bandwidth
*   **Bandwidth:** Theoretical maximum speed of a link.
*   **Throughput:** Actual speed achieved in real-world conditions (influenced by congestion).

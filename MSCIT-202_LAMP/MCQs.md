# MSCIT-202: Web Application Development – LAMP
## Important MCQs from Previous Papers

1. **_______ string function is used to remove spaces from the string.** (Aug 2022)
   A. cut() B. trim() C. remove() D. del_sp()
   **Answer: B.** `trim()` removes whitespace from both sides of a string.

2. **To find first occurrence of the string inside a string with case insensitive manner, _____ function is used.** (Aug 2022)
   A. strstr() B. find() C. stristr() D. search()
   **Answer: C.** `stristr()` is case-insensitive.

3. **To convert first letter of each word into upper case ______ function is used.** (Aug 2022)
   A. ucwords() B. lcfirst() C. ucfirst() D. None
   **Answer: A.** `ucwords()` capitalizes the first character of each word.

4. **If the existing file with data is opened to write _____ file mode is used.** (Aug 2022)
   A. a B. r C. w D. z
   **Answer: C.** `w` opens for writing only; truncates file to zero length.

5. **_______ command is used to change the directory in Linux.** (Aug 2022)
   A. chngdir() B. chdir() C. getcwd() D. None
   **Answer: B.** `chdir()` in PHP; `cd` in Linux shell.

6. **______ function is used to move file pointer from its current location to new location.** (Aug 2022)
   A. fclose() B. fopen() C. ftell() D. fseek()
   **Answer: D.** `fseek()` moves the pointer.

7. **To destroy session item, _______ function is used.** (Aug 2022)
   A. session_destroy() B. session_end() C. unset() D. free()
   **Answer: A.** (Note: `unset()` removes a specific variable, `session_destroy()` destroys all).

8. **To sort an array by values, using a user defined comparison function, ____ function is used.** (Aug 2022)
   A. uasort() B. uksort() C. rsort() D. prev()
   **Answer: A.** `uasort()` sorts by values with user-defined function.

9. **If we want a day of the month without leading 0, then ____ parameter is used with date function.** (Aug 2022)
   A. d B. j C. D D. k
   **Answer: B.** `j` is day of month without leading zeros.

10. **To change the file permission in the Linux, ______ command is used.** (Aug 2022, July 2023)
    A. change B. chown C. fperm D. chmod
    **Answer: D.** `chmod` stands for change mode.

11. **_______ software is a kind of computer software whose source code is made available to all the users.** (Jan 2023)
    A. Close B. Open source C. Copy right D. Licensed
    **Answer: B.**

12. **Which of the following is an open source Operating System?** (Jan 2023)
    A. Linux B. Android C. Ubuntu D. All
    **Answer: D.**

13. **According to GPL, a software is called free software if its users have freedom to:** (Jan 2023)
    A. Run it for any purpose B. Share it with others C. Change it D. All
    **Answer: D.**

14. **The _________ command in Linux is used to find a file.** (Jan 2023)
    A. find B. search C. locate D. Both A & C
    **Answer: D.**

15. **The Linux kernel tracks every aspect of a process by its ____** (Jan 2023)
    A. BID B. PID C. CID D. TID
    **Answer: B.** PID stands for Process ID.

16. **In Linux ________ command is used to terminate all the running processes.** (Jan 2023)
    A. Terminate B. Remove C. Kill D. Delete
    **Answer: C.**

17. **SQL consists of ___________ built-in sub-languages.** (Jan 2023)
    A. Two B. Three C. Four D. Five
    **Answer: C.** (DDL, DML, DCL, TCL)

18. **_________ integrity enforces valid entries for a given column by restricting type/format/range.** (July 2023)
    A. Domain B. Referential C. User-defined D. Entity
    **Answer: A.**

19. **___________ join returns records that have matching values in both tables.** (July 2023)
    A. Left B. Right C. Full D. Inner
    **Answer: D.**

20. **Which of the following is NOT an open source operating system?** (July 2023, July 2025)
    A. Ubuntu B. Windows C. Linux D. Fedora
    **Answer: B.** Windows is Proprietary/Closed-source.

21. **In Linux file permissions, "rwx" stands for:** (July 2025)
    A. read, write, execute B. run, write, execute C. read, work, execute D. read, write, expand
    **Answer: A.**

22. **Which loop in PHP is best for executing code when the number of iterations is unknown?** (July 2025)
    A. for B. while C. do-while D. foreach
    **Answer: B.** `while` is used for conditional loops.

23. **Which function in PHP is used to count elements in an array?** (July 2025)
    A. sizeof() B. length() C. count() D. Both A & C
    **Answer: D.** Both `count()` and `sizeof()` are used.

24. **Which PHP superglobal is used to collect form data sent via GET method?** (July 2025)
    A. $_POST B. $_GET C. $_REQUEST D. $_FORM
    **Answer: B.**

25. **Which function in PHP is used to connect to a MySQL database using MySQLi?** (July 2025)
    A. mysqli_connect() B. mysql_connect() C. connect_mysql() D. db_connect()
    **Answer: A.**

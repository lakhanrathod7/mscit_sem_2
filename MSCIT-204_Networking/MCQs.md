# MSCIT-204: Fundamental of Computer Networking
## Important MCQs from Previous Papers

1. **Header of a frame generally contains ______________** (Aug 2022)
   A. synchronization bytes B. addresses C. frame identifier D. all
   **Answer: D.**

2. **The network layer is concerned with __________ of data.** (Aug 2022)
   A. bits B. frames C. packets D. bytes
   **Answer: C.** (Bits-Physical, Frames-DataLink, Packets-Network).

3. **Which one of the following algorithms is NOT used for congestion control?** (Aug 2022)
   A. traffic aware routing B. admission control C. load shedding D. routing information protocol
   **Answer: D.** RIP is a routing protocol, not a congestion control algorithm.

4. **ICMP is primarily used for __________** (Aug 2022)
   A. error and diagnostic functions B. addressing C. forwarding D. routing
   **Answer: A.**

5. **Which of the following are transport layer protocols used in networking?** (Aug 2022)
   A. TCP and FTP B. UDP and HTTP C. TCP and UDP D. HTTP and FTP
   **Answer: C.**

6. **User datagram protocol (UDP) is called connectionless because _____________** (Aug 2022)
   A. all UDP packets are treated independently B. it sends as stream C. received in same order D. very quick
   **Answer: A.**

7. **A _____ is a TCP name for a transport service access point.** (Aug 2022)
   A. port B. pipe C. node D. protocol
   **Answer: A.**

8. **Physical or logical arrangement of network is __________** (Aug 2022, July 2025)
   A. Routing B. Topology C. Control D. Networking
   **Answer: B.**

9. **Multiplexing is used in _______** (Aug 2022, Jan 2023)
   A. Packet switching B. Circuit switching C. Data switching D. Both A & B
   **Answer: D.**

10. **Communication channel is shared by all the machines on the network in ________** (Aug 2022)
    A. broadcast network B. unicast network C. multicast network D. anycast network
    **Answer: A.**

11. **Which of the following performs modulation and demodulation?** (Jan 2023)
    A. Modem B. Coaxial cable C. Fiber optics D. None
    **Answer: A.**

12. **In TDM, slots are further divided into______.** (Jan 2023)
    A. Seconds B. Frames C. Packets D. Bits
    **Answer: B.**

13. **The loss in signal power as light travels down the fiber is called _____.** (Jan 2023, July 2023, July 2025)
    A. Attenuation B. Propagation C. Scattering D. Interruption
    **Answer: A.**

14. **Identify the device used to boost up a weak signal.** (Jan 2023, July 2025)
    A. Modem B. Repeater C. Switch D. Router
    **Answer: B.**

15. **Network providing a high speed connectivity is:** (July 2023)
    A. MAN B. LAN C. WAN D. Internetwork
    **Answer: B.**

16. **A transmission media can have signal impairment because of:** (July 2023)
    A. Noise B. Attenuation C. distortion D. All
    **Answer: D.**

17. **Which of the following topologies requires a multi-point connection?** (July 2023)
    A. Ring B. Star C. Mesh D. Bus
    **Answer: D.**

18. **In a _________ network a couple of computers are connected through media to share resources temporarily.** (July 2025)
    A. Peer to Peer B. Client Server C. Distributed D. All
    **Answer: A.**

19. **_________ noise is irregular pulses or noise spikes in short time.** (July 2025)
    A. Induced B. Crosstalk C. Thermal D. Impulse
    **Answer: D.**

20. **Time required for a bit to travel from source to destination is called _________ Time.** (July 2025)
    A. Latency B. Propagation C. Jitter D. Transmission
    **Answer: B.**

# MSCIT-204: Fundamental of Computer Networking
## Important MCQs from Previous Papers

1. Header of a frame generally contains ______________ (Aug 2022)
   A. synchronization bytes B. addresses C. frame identifier D. all of the mentioned
   Answer: D.
   Frame headers include sync bits, source/dest addresses, and control info.

2. The network layer is concerned with __________ of data. (Aug 2022)
   A. bits B. frames C. packets D. bytes
   Answer: C.
   Network layer encapsulates data into packets (IP datagrams).

3. Which one of the following algorithm is not used for congestion control? (Aug 2022)
   A. traffic aware routing B. admission control C. load shedding D. routing information protocol
   Answer: D.
   RIP is a distance-vector routing protocol, not a congestion control mechanism.

4. ICMP is primarily used for __________ (Aug 2022)
   A. error and diagnostic functions B. addressing C. forwarding D. routing
   Answer: A.
   Internet Control Message Protocol reports errors and helps in diagnostics (like ping).

5. Which of the following are transport layer protocols used in networking? (Aug 2022)
   A. TCP and FTP B. UDP and HTTP C. TCP and UDP D. HTTP and FTP
   Answer: C.
   TCP and UDP are the two primary transport layer protocols.

6. User datagram protocol is called connectionless because _____________ (Aug 2022)
   A. all UDP packets are treated independently B. it sends data as a stream C. received in same order D. sends data very quickly
   Answer: A.
   Each UDP packet is routed independently without a pre-established connection.

7. A _____ is a TCP name for a transport service access point. (Aug 2022)
   A. port B. pipe C. node D. protocol
   Answer: A.
   Ports identify specific processes or services on a host.

8. Physical or logical arrangement of network is __________ (Aug 2022)
   A. Routing B. Topology C. Control D. Networking
   Answer: B.
   Topology describes how nodes are interconnected.

9. Multiplexing is used in _______ (Aug 2022)
   A. Packet switching B. Circuit switching C. Data switching D. Packet & Circuit switching
   Answer: D.
   Both switching methods utilize multiplexing to share channel capacity.

10. Communication channel is shared by all the machines on the network in ________ (Aug 2022)
    A. broadcast network B. unicast network C. multicast network D. anycast network
    Answer: A.
    In broadcast, every packet is received by every node on the network.

11. Which of the following layers is an addition to OSI model when compared with TCP/IP model? (Jan 2023)
    A. Application layer B. Presentation layer C. Session layer D. Session and Presentation layer
    Answer: D.
    TCP/IP collapses OSI's Session, Presentation, and Application layers into one.

12. Which network topology requires a central controller or hub? (Jan 2023)
    A. Star B. Mesh C. Ring D. Bus
    Answer: A.
    In a Star topology, all devices connect to a central hub/switch.

13. Which of the following performs modulation and demodulation? (Jan 2023)
    A. Modem B. Coaxial cable C. Fiber optics D. None of these
    Answer: A.
    Modem stands for MOdulator-DEModulator.

14. In TDM, slots are further divided into______. (Jan 2023)
    A. Seconds B. Frames C. Packets D. Bits
    Answer: B.
    Time Division Multiplexing organizes slots into repeating frames.

15. Multiplexing is used in ______. (Jan 2023)
    A. Packet switching B. Circuit switching C. Data switching D. Packet and Circuit switching
    Answer: D.
    Multiplexing allows multiple signals to share a single physical link.

16. The loss in signal power as light travels down the fiber is called _____. (Jan 2023)
    A. Attenuation B. Propagation C. Scattering D. Interruption
    Answer: A.
    Attenuation is the reduction in signal strength over distance.

17. Identify among the following which is mainly used to host web site. (Jan 2023)
    A. Mail server B. Web server C. Database server D. None of these
    Answer: B.
    Web servers (like Apache) host and serve website files.

18. Identify the total versions of IP. (Jan 2023)
    A. 1 B. 2 C. 3 D. 4
    Answer: B.
    The two primary versions are IPv4 and IPv6.

19. Identify the device used to boost up a weak signal. (Jan 2023)
    A. Modem B. Repeater C. Switch D. Router
    Answer: B.
    Repeaters regenerate and amplify signals for longer distances.

20. Which of the following switching method message is divided into small packets? (Jan 2023)
    A. Virtual switching B. Message switching C. Packet switching D. None of these
    Answer: C.
    Packet switching breaks messages into smaller chunks called packets.

21. Network providing a high speed connectivity is (July 2023)
    A. MAN B. LAN C. WAN D. Internetwork
    Answer: B.
    LANs (Local Area Networks) typically provide high speeds for small areas.

22. Mail access starts with client when user needs to download e-mail from the (July 2023)
    A. Mail Box B. Mail Server C. Mail Host D. Internet
    Answer: B.
    Clients connect to a mail server (e.g., via POP3/IMAP) to get messages.

23. A transmission media can have signal impairment because of (July 2023)
    A. Noise B. Attenuation C. distortion D. All of these
    Answer: D.
    All three (attenuation, distortion, noise) degrade signal quality.

24. Completion of one full pattern is called a (July 2023)
    A. Period B. Cycle C. Frame D. Segment
    Answer: B.
    A cycle is one complete set of values in a periodic signal.

25. Propagation time is equals to (July 2023)
    A. Distance/Propagation speed B. Propagation speed/Bandwidth C. Message size/ Bandwidth D. Bandwidth/Queuing time
    Answer: A.
    It is the time taken for a bit to travel from sender to receiver.

26. The loss of strength of a signal while propagation through a medium is (July 2023)
    A. Attenuation B. Distortion C. Noise D. Impairments
    Answer: A.
    Signal energy decreases as it travels through a medium.

27. The techniques for efficient utilization of bandwidth includes (July 2023)
    A. Medium linking B. Sampling C. Multiplexing D. None of these
    Answer: C.
    Multiplexing allows multiple users to share the same bandwidth.

28. The data link layer takes packets from _________ and encapsulates them into frames. (July 2023)
    A. network layer B. physical layer C. transport layer D. application layer
    Answer: A.
    DLL receives packets from the layer above (Network layer).

29. Which of the following topologies requires a multi-point connection? (July 2023)
    A. Ring B. Star C. Mesh D. Bus
    Answer: D.
    In a Bus topology, all nodes share a single common cable.

30. Which of the following layers does the HTTP protocol work on? (July 2023)
    A. Physical layer B. Data-link layer C. Application layer D. None of these
    Answer: C.
    HTTP is an application layer protocol for transferring web data.

31. In a _________ network a couple of computers are connected to share resources temporarily. (July 2025)
    A. Peer to Peer B. Client Server C. Distributed D. All of them
    Answer: A.
    P2P involves direct resource sharing without a central server.

32. _________ noise is irregular pulses or noise spikes in short time. (July 2025)
    A. Induced B. Crosstalk C. Thermal D. Impulse
    Answer: D.
    Impulse noise is caused by external disturbances like lightning.

33. Time required for a bit to travel from source to destination is called _________ Time. (July 2025)
    A. Latency B. Propagation C. Jitter D. Transmission
    Answer: B.
    Propagation time = Distance / Speed.

34. _________ provides support for Internet standard process. (July 2025)
    A. Internet Society (ISOC) B. Internet Architecture Board(IAB) C. IETF D. All of them
    Answer: D.
    Multiple organizations collaborate to maintain internet standards.

35. Application layer defines _________. (July 2025)
    A. Message format B. Syntax C. Semantics D. All of them
    Answer: D.
    Application layer standards cover formats, rules, and meanings of data.

36. IEEE _________ defines Wireless Local Area Network. (July 2025)
    A. 802.15 B. 802.11 C. 802.3 D. 802.5
    Answer: B.
    802.11 is the standard for Wi-Fi.

37. A static IP address is primarily used for smaller networks and supports up to 254 devices. (July 2025)
    A. Class A B. Class B C. Class C D. Class D
    Answer: C.
    Class C allows for 254 hosts (e.g., 192.168.1.1 to 192.168.1.254).

38. _________ is operates at data link layer. (July 2025)
    A. Bridge B. Gateway C. Router D. Hub
    Answer: A.
    Bridges (and Switches) operate at Layer 2 (Data Link).

39. In _________ method or random-access method, no station is superior to another. (July 2025)
    A. Polling B. Token Passing C. Contention D. All of them
    Answer: C.
    In contention (like ALOHA), nodes compete for channel access.

40. In _________ topology, each computer is connected to exactly two neighbors. (July 2025)
    A. Ring B. Mesh C. Star D. Bus
    Answer: A.
    In a Ring, each node has two connections forming a closed loop.

# MSCIT-203: Object Oriented Analysis and Design using UML
## Important MCQs from Previous Papers

1. **Interaction diagrams are divided into _________ main types of diagrams.** (Aug 2022)
   A. Three B. Four C. Five D. Six
   **Answer: B.** (Sequence, Communication, Timing, Interaction Overview).

2. **__________ message waits for a reply before the interaction can move forward.** (Aug 2022)
   A. Synchronous B. Asynchronous C. Delete D. Self
   **Answer: A.**

3. **____________ operator is allowed within an interaction diagram.** (Aug 2022)
   A. Opt B. Alt C. Ref D. All
   **Answer: D.**

4. **_____ is the idea of focusing on the common properties and behaviors of the object ignoring irrelevant details.** (Aug 2022)
   A. Encapsulation B. Inheritance C. Abstraction D. None
   **Answer: C.**

5. **__________ is the ability to take multiple forms.** (Aug 2022)
   A. Polymorphism B. Inheritance C. Abstraction D. Encapsulation
   **Answer: A.**

6. **What is full form of CRC?** (Aug 2022, Jan 2023)
   A. Class-Responsibility-Collaborator B. Class-Responsibility-Contractor C. Class-Responsibility-Collection D. Class-Responsibility-Combination
   **Answer: A.**

7. **_______ is an extension of the information hiding concept.** (Aug 2022)
   A. Coupling B. Cohesion C. Modularity D. None
   **Answer: C.**

8. **A design pattern used to enhance the functionality of an object is:** (Jan 2023)
   A. Adapter B. Decorator C. Delegation D. Proxy
   **Answer: B.**

9. **Which web app attribute is defined by: “A large number of users may access the WebApp at one time”?** (Jan 2023)
   A. Unpredictable load B. Performance C. Concurrency D. Network intensiveness
   **Answer: C.**

10. **_____ shows the interactions between objects or parts in terms of sequenced messages.** (Jan 2023)
    A. Sequence diagram B. Interaction overview diagram C. Timing Diagram D. Communication Diagram
    **Answer: A.**

11. **______________ design patterns deal with object creation mechanism.** (July 2023)
    A. Structural B. Behavioral C. Creational D. None
    **Answer: C.**

12. **In web design pyramid _______ design describes the overall "look and feel" of the WebApp.** (July 2023, July 2025)
    A. Aesthetic B. Interface C. Content D. Component
    **Answer: A.** (Aesthetic design is also known as "Graphics" or "Look and Feel").

13. **A country has a capital is an example of _________ type of relationship.** (July 2025)
    A. Many to One B. One to One C. One to Many D. Many to Many
    **Answer: B.**

14. **The ________________ model consists of structural elements.** (July 2025)
    A. Content B. Interaction C. Functional D. Configuration
    **Answer: A.** (Content model represents structural elements like data objects).

15. **_________ diagrams express both the context of a group of objects and the interaction between them.** (July 2025)
    A. Deployment B. Sequence C. Use case D. Collaboration
    **Answer: D.**

16. **__________ are those who interact with the system (human or other system).** (July 2025)
    A. Connections B. Actors C. Use cases D. All
    **Answer: B.**

17. **__________ diagram shows the configuration of run-time processing nodes.** (July 2025)
    A. Deployment B. Collaboration C. Statecraft D. Component
    **Answer: A.**

18. **A design pattern often used to restrict access to an object is:** (Jan 2023, July 2025)
    A. Proxy B. Adapter C. Decorator D. Delegation
    **Answer: A.**

19. **_________ is a process of information (data) hiding.** (July 2025)
    A. Abstraction B. Polymorphism C. Encapsulation D. Inheritance
    **Answer: C.**

20. **UML Stands for _____________** (Jan 2023, July 2023)
    A. Uniform Modeling Language B. Universal Modeling Language C. United Modeling Language D. Unified Modeling Language
    **Answer: D.**

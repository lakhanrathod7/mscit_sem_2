# MSCIT-203: Object Oriented Analysis and Design using UML
## Important MCQs from Previous Papers

1. Interaction diagrams are divided into _________ main types of diagrams. (Aug 2022)
   A. Three B. Four C. Five D. Six
   Answer: B.
   They include Sequence, Communication, Timing, and Interaction Overview diagrams.

2. __________ message waits for a reply before the interaction can move forward. (Aug 2022)
   A. Synchronous B. Asynchronous C. Delete D. Self
   Answer: A.
   Synchronous messages represent a blocking call.

3. ____________ operator is allowed within an interaction diagram. (Aug 2022)
   A. Opt B. Alt C. Ref D. All of these
   Answer: D.
   These operators are used in interaction fragments to model logic.

4. _____is the idea of focusing on the common properties and behaviours of the object and ignore what’s unimportant. (Aug 2022)
   A. Encapsulation B. Inheritance C. Abstraction D. None of these
   Answer: C.
   Abstraction simplifies complex systems by modeling essential features.

5. __________ is originally a Greek word that means the ability to take multiple forms. (Aug 2022)
   A. Polymorphism B. Inheritance C. Abstraction D. Encapsulation
   Answer: A.
   Polymorphism allows a single interface to represent multiple underlying forms.

6. Object Oriented System Development Life Cycle consists of: (Aug 2022)
   A. Analysis B. Design C. Implementation D. All of these
   Answer: D.
   OO SDLC includes analysis, design, and implementation stages.

7. The data model consists of ______ interconnected pieces of information. (Aug 2022)
   A. Two B. Three C. Four D. Five
   Answer: B.
   Typically data objects, attributes, and relationships.

8. _____ is the data model must be capable of representing the number of occurrences of objects. (Aug 2022)
   A. Modality B. Multiplicity C. Cardinality D. None of these
   Answer: C.
   Cardinality defines the numeric relationship between objects.

9. What is full form of CRC? (Aug 2022)
   A. Class-Responsibility-Collaborator B. Class-Responsibility-Contractor C. Class-Responsibility-Collection D. Class-Responsibility-Combination
   Answer: A.
   CRC cards are used to brainstorm object designs.

10. _______ is an extension of the information hiding concept. (Aug 2022)
    A. Coupling B. Cohesion C. Modularity D. None of these
    Answer: C.
    Modularity divides a system into independent, exchangeable units.

11. Identify the correct statements regarding Coupling and Cohesion. (Jan 2023)
    A. Strongly coupled classes produce better design B. Coupling is degree of interdependence C. Cohesion is functional strength D. Both B and C
    Answer: D.
    Good design requires low coupling and high cohesion.

12. Use Case diagrams are composed of? (Jan 2023)
    A. Uses, cases B. People, classes and objects C. People, computer D. Actors, use cases
    Answer: D.
    Actors represent users and Use Cases represent system functions.

13. A design pattern used to enhance the functionality of an object is: (Jan 2023)
    A. Adapter B. Decorator C. Delegation D. Proxy
    Answer: B.
    Decorator pattern dynamically adds responsibilities to objects.

14. A design pattern often used to restrict access to an object is ___________ (Jan 2023)
    A. Adapter B. Decorator C. Delegation D. Proxy
    Answer: D.
    Proxy acts as a placeholder or gatekeeper for another object.

15. Which web app attribute is defined by: “A large number of users may access the WebApp at one time”? (Jan 2023)
    A. Unpredictable load B. Performance C. Concurrency D. Network intensiveness
    Answer: C.
    Concurrency deals with multiple simultaneous user requests.

16. Which process model should be used in virtually all situations of web engineering? (Jan 2023)
    A. Incremental Model B. Waterfall Model C. Spiral Model D. None of the mentioned
    Answer: A.
    Incremental development suits the evolving nature of web apps.

17. Which of the following statements are incorrect with reference to web based systems? (Jan 2023)
    A. Web-based systems should be unscalable B. They must cope with random demands C. They must be secure D. They are subject to legal scrutiny
    Answer: A.
    Scalability is a mandatory requirement for web systems.

18. What category of web-based system would you assign to electronic shopping? (Jan 2023)
    A. Informational B. Interactive C. Transaction-oriented D. Workflow-oriented
    Answer: C.
    E-commerce involves database transactions.

19. UML Stands for _____________ (Jan 2023)
    A. Uniform Modeling Language B. Universal Modeling Language C. United Modeling Language D. Unified Modeling Language
    Answer: D.
    Unified Modeling Language is the industry standard for modeling.

20. _____ shows the interactions between objects or parts in terms of sequenced messages. (Jan 2023)
    A. Sequence diagram B. Interaction overview diagram C. Timing Diagram D. Communication Diagram
    Answer: A.
    Sequence diagrams emphasize the chronological order of messages.

21. ______________ design patterns deals with object creation mechanism. (July 2023)
    A. Structural B. Behavioral C. Creational D. None of them
    Answer: C.
    Creational patterns abstract the instantiation process.

22. Use-cases model the system from the ____________ point of view. (July 2023)
    A. Analyst’s B. End-user’s C. Programmer's D. Tester’s
    Answer: B.
    Use cases capture what the user expects from the system.

23. ______________ web application category is where the user queries databases and extracts information. (July 2023)
    A. Portal B. Service-Oriented C. Transaction-Oriented D. Data warehousing
    Answer: D.
    Data warehousing involves complex queries on large data sets.

24. _____________is one that is stored on a server and delivered over the Internet through a browser. (July 2023)
    A. Web Application B. Website C. Search Engine D. Web Crawler
    Answer: A.
    Web apps provide interactive functionality via the web.

25. The ________________ model is created from use-cases. (July 2023)
    A. Content B. Interaction C. Functional D. Configuration
    Answer: B.
    Functional model describes the behavior required by use cases.

26. In web design pyramid _______ design describes the overall "look and feel" of the WebApp. (July 2023)
    A. Aesthetic B. Interface C. Content D. Component
    Answer: A.
    Aesthetic design focuses on visual appeal (Graphics).

27. __________design represents the path between content objects and for each category of users. (July 2023)
    A. Content B. Architecture C. Aesthetic D. Navigation
    Answer: D.
    Navigation design defines how users move through the app.

28. ____________ diagram describes hardware, execution environments, and artifacts. (July 2023)
    A. Deployment B. Component C. Class D. Object
    Answer: A.
    Deployment diagrams show the physical topology of the system.

29. A tutorial or e-book presentation is a simple example of _________ web app structure. (July 2023)
    A. Grid B. Hierarchical C. Linear D. All of them
    Answer: C.
    Linear structure follows a sequential path.

30. UML is largely the product of well known software engineer ____________ (July 2023)
    A. Grady Booch B. James Rumbaugh C. Ivar Jacobson D. All of them
    Answer: D.
    Known as the "Three Amigos", they merged their methods into UML.

31. A design pattern often used to restrict access to an object is ___________ (July 2025)
    A. Proxy B. Adapter C. Decorator D. Delegation
    Answer: A.
    Proxy serves as a gatekeeper to the real object.

32. ____________ is a process of information (data) hiding. (July 2025)
    A. Abstraction B. Polymorphism C. Encapsulation D. Inheritance
    Answer: C.
    Encapsulation bundles data and methods, hiding internal details.

33. A country has a capital is an example of _________ type of relationship. (July 2025)
    A. Many to One B. One to One C. One to Many D. Many to Many
    Answer: B.
    Each country has exactly one capital and each capital belongs to one country.

34. _______ design is also known as "Look and Feel". (July 2025)
    A. Content B. Architecture C. Graphics D. Aesthetic
    Answer: D.
    Aesthetic design is the visual layer of the web app.

35. The ________________ model consists of structural elements. (July 2025)
    A. Content B. Interaction C. Functional D. Configuration
    Answer: A.
    Content model maps out the information objects.

36. __________ diagram describes the internal structure of a class and the collaborations it makes possible. (July 2025)
    A. Deployment B. Package C. Composite Structure D. Component
    Answer: C.
    Composite structure diagrams show the internal parts and ports of a class.

37. A ___________ message is used to represent a scenario where an unknown source sends the message. (July 2025)
    A. Lost B. Found C. Self D. Asynchronous
    Answer: B.
    Found messages come from an unspecified source outside the scope.

38. _________ diagrams express both the context of a group of objects and the interaction between them. (July 2025)
    A. Deployment B. Sequence C. Use case D. Collaboration
    Answer: D.
    Collaboration (Communication) diagrams emphasize object organization.

39. ___________ are those who interact with the system that is human or other system. (July 2025)
    A. Connections B. Actors C. Use cases D. All of them
    Answer: B.
    Actors are entities external to the system that play a role.

40. __________ diagram shows the configuration of run-time processing nodes and components. (July 2025)
    A. Deployment B. Collaboration C. Statecraft D. Component
    Answer: A.
    Deployment diagrams map software artifacts to hardware nodes.
